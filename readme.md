# Winter Burrow Zoom Mod (BepInEx)

A lightweight camera zoom mod for *Winter Burrow*, built using **BepInEx 5**.  
Allows smooth zooming in and out with keyboard keys.

## Installation
1. Install **BepInEx 5 (x64)** for Winter Burrow.  
2. Copy the file **WinterBurrowZoomMod.dll** into the folder: BepInEx/plugins inside your game directory.  


## Controls
- **+** : Zoom in  
- **-** : Zoom out  
- **0** : Reset zoom  

## Features
- Works only on the main world camera  
- Smooth FOV-based zoom  
- No configuration needed  
- Safe, does not modify game assets

## Uninstall
Delete **WinterBurrowZoomMod.dll** from the BepInEx/plugins folder.
